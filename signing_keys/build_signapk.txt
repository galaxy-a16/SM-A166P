How to build?

    git clone https://github.com/salvogiangri/signapk.git
    cd signapk
    git submodule update --init --recursive
    ./gradlew build

output file: signapk/signapk/build/libs/signapk-all.jar

usage:

java -jar signapk.jar aosp_platform.x509.pem aosp_platform.pk8 KmxService_new.apk KmxService_new_signed.apk
